## Overview

A ridiculously simple message board.  This was a five minute Express practice 
project.

The full design requirements from The Odin Project to build a message board app
are [here](https://www.theodinproject.com/paths/full-stack-javascript/courses/nodejs/lessons/mini-message-board).
